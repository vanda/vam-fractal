# Unstyled input (mixin)

## Background and usage

Resets an input (`button` or `a`) to appeared unstyled

## Implementation

```sass
.foo{
  @include unstyled-input;
}
```

# Unstyled list (mixin)

## Background and usage

Resets a list (`ol` or `ul`) to appeared unstyled

## Implementation

```sass
.foo{
  @include unstyledList;
}
```
