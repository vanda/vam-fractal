Light/Dark Theme

Light theme is default as per common design patterns.

Some basic light/dark colors are setup globally as CSS Custom Properties.

Components should be coded to respond to light/dark colors using their own additional CSS Custom Properties as needed. e.g. Breadcrumb component

Dark theme can be applied as an override using s-light-dark-theme--dark class on any element and will be inherited by its children.

Or, Auto theme can be applied using s-light-dark-theme--auto which will similarly override Light to Dark theme when System prefs are set to dark.
